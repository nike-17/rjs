Родной Земли

RJS преобразует исконно русские повеления в машинный код иноземцев (ES6), сохраняя дух и логику разработчика.

Шаг I. Написание

Излагайте мысли на могучем русском языке.

Шаг II. Трансляция

Мгновенный перевод без потери смысла.

Шаг III. Исполнение

Работает везде: от браузера до 1С.

- Исходный Текст -
константа Империя = истина;
переменная казна = 1000;

функция собратьНалог(сумма) {
  если (казна < 5000) {
    казна += сумма;
    консоль.лог("Казна пополнена: " + сумма);
  } иначе {
    консоль.лог("Казна полна, милорд!");
  }
  вернуть казна;
}

// Ежегодный сбор
пока (казна < 3000) {
  собратьНалог(500);
}

                    Вайб Кодинга на Русском
                

                

                

                    Когда код звучит как родная речь, программирование становится искусством
                

            


            
            

                
                

                    

                        

                            

                                
                            

                            

                                
Думай на Родном

                                

                                    Не нужно переводить мысли с русского на английский и обратно.
                                    Пиши «если» вместо if,
                                    «функция» вместо function.
                                

                            

                        

                        

                            «Когнитивная нагрузка падает, код пишется сам собой»
                        

                    


                    

                        

                            

                                
                            

                            

                                
Радость от Процесса

                                

                                    Код читается как книга. Переменные «пользователь»,
                                    «корзина», «заказ» —
                                    всё понятно с первого взгляда.
                                

                            

                        

                        

                            «Программирование перестало быть работой — стало творчеством»
                        

                    


                    

                        

                            

                                
                            

                            

                                
Доступно Всем

                                

                                    Бабушка-бухгалтер, школьник, физрук — все могут писать код.
                                    Не нужно учить английский, чтобы автоматизировать свою работу.
                                

                            

                        

                        

                            «Наконец-то программирование для людей, а не для избранных»
