# Russian JS Transpiler

**Russian JS** (RJS) — это инструмент, который позволяет писать JavaScript-код на русском языке.

## Начало работы

1. [Установка](./getting-started)
2. [Синтаксис](./syntax)
3. [API](./api)

## Примеры

```javascript
функция приветствие(имя) {
  вернуть `Привет, ${имя}!`;
}

консоль.лог(приветствие('Мир')); // Выведет: Привет, Мир!
```
