# Добро пожаловать в Russian JS Transpiler

Russian JS Transpiler (RJS) — это инструмент, который позволяет писать JavaScript-код на русском языке. 
С RJS вы можете использовать русские ключевые слова, имена переменных и функций, что делает 
программирование более доступным для русскоязычных разработчиков.

## Быстрый старт

Установите RJS глобально:

```bash
npm install -g russian-js-transpiler
```

Или как dev-зависимость в ваш проект:

```bash
npm install --save-dev russian-js-transpiler
```

## Простой пример

```rjs
// Файл: приветствие.ржс
функция приветствовать(имя) {
  вернуть `Привет, ${имя}!`;
}

консоль.лог(приветствовать("Мир")); // Выведет: Привет, Мир!
```

Скомпилируйте файл:

```bash
rjs compile приветствие.ржс -o приветствие.js
```

## Возможности

- Полная поддержка современного JavaScript (ES6+)
- Русские ключевые слова и синтаксис
- Интеграция с существующими JavaScript-библиотеками
- Поддержка TypeScript
- Гибкая настройка через конфигурационный файл
